## Setup
To get the code working, follow these steps:

**build containers**

Before running the makefile, make sure you have (Docker, Docker-compose) installed

```
make all
```

**Display useful make commands**

```
make help
```

Now check out the site at `http://localhost:8000`

Have fun!
